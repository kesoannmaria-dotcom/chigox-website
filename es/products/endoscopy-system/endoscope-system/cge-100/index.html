Resumen

Descripción del producto

CGE-100 es una plataforma de endoscopia médica Full HD 1080P diseñada para procedimientos gastrointestinales, respiratorios, ORL, urológicos y de endoscopia rígida. El sistema incluye procesador de video, fuente de luz fría LED, monitor y software de estación de trabajo opcional. Un procesador puede admitir varios endoscopios flexibles y rígidos, ofreciendo una solución escalable para hospitales, clínicas y centros de endoscopia.

Características principales

  • El sistema CGE-100 admite una estación de trabajo médica opcional para gestión de datos y documentación clínica eficiente.
  • Gestión de información del paciente
  • Captura de imágenes y video
  • Registro de procedimientos
  • Gestión de base de datos de casos
  • Generación de informes
  • Anotación de imágenes
  • Exportación de datos USB
