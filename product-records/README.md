# Protected Product Records

This directory contains the authoritative source records for product pages that
must not be regenerated from guesses, placeholders, or unrelated product data.

Before editing a product page:

1. Check whether a matching folder exists here.
2. Preserve the recorded product images and specifications.
3. Update the record and the website page together.
4. Do not replace recorded content during bulk translation or page generation.

These records are intentional source archives, not redundant deployment files.
