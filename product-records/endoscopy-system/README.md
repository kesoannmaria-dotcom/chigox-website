## Protection Rules

- Do not replace these products with veterinary endoscopy content.
- Use the supplied disposable image for CGU-650D and the supplied reusable
  image for CGU-750R. Do not swap these images between the two products.
- Keep technical specifications as real responsive tables.
- Preserve reusable and disposable products as separate product pages.
- Display disposable and reusable products in separate columns on category pages
  and as separate links in the Products navigation menu.
- Always display Reusable products before Disposable products.
- Use this category order and spelling: Endoscope System, Bronchoscope,
  Cystoscope, Nasopharyngoscope, Ureteroscope, Rigid endoscope.
- In the navigation menu, show descriptive product-type names rather than model
  numbers: Reusable/Disposable Bronchoscope, Cystoscope, Nasopharyngoscope and
  Ureteroscope; use Rigid Endoscope Series for the rigid product.
- Bronchoscope must contain separate Reusable and Disposable product pages.
- The supplied CGB-600 document contains reusable `CGB-600R` models only. Do
  not copy those specifications to the disposable product.
- Check this archive before bulk generation, translation, navigation updates, or
  product-page cleanup.
- Update both this archive and the corresponding website page when source
  information changes.
